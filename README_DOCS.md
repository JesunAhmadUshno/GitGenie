## 🛠️ Technology Why-Choices

### Frontend
- **Next.js 14**: SSR/SSG performance, built-in optimizations
- **React 18**: Component-based UI, ecosystem
- **TypeScript**: Type safety, developer experience
- **Tailwind CSS**: Rapid styling, design consistency
- **Framer Motion**: Beautiful animations, performance

### Backend
- **Express.js**: Lightweight, flexible, battle-tested
- **PostgreSQL**: Relational data, robust
- **Redis**: Real-time features, caching, leaderboards
- **Prisma ORM**: Type-safe DB access, migrations

### Deployment
- **Vercel**: Next.js optimized, auto-scaling
- **Railway**: Simple backend hosting, affordable
- **GitHub Actions**: Native CI/CD integration

# GitGenie

Unlock and track GitHub profile achievements with a modern interactive dashboard.

## Live Demo

- https://jesunahmadushno.github.io/GitGenie

## Why GitGenie?

GitGenie helps developers understand and pursue GitHub achievements through a polished UI, clear progress views, and practical guidance.

## Features

- Achievement-focused dashboard experience
- Progress-oriented visual sections for common badges
- GitHub Pages-compatible static deployment
- Responsive, modern interface built with Next.js

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- GitHub Actions

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

GitGenie is configured for GitHub Pages deployment via workflow:

- `.github/workflows/deploy-pages.yml`
